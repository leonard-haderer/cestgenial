# Sources des Crises Documentées

Ce document liste les sources utilisées pour les crises réelles documentées dans le fichier `Base_Crises_TresTres_Enrichie_CGenial.csv`.

## Sources Générales

### Bases de données officielles
- **EM-DAT (Emergency Events Database)** : Base de données internationale sur les catastrophes naturelles et technologiques (www.emdat.be)
- **USGS (United States Geological Survey)** : Données sur les séismes (earthquake.usgs.gov)
- **NOAA (National Oceanic and Atmospheric Administration)** : Données sur les ouragans et cyclones (www.noaa.gov)
- **OCHA (Office for the Coordination of Humanitarian Affairs)** : Données sur les crises humanitaires (www.unocha.org)
- **ACLED (Armed Conflict Location & Event Data Project)** : Données sur les conflits armés (acleddata.com)

### Organisations internationales
- **ONU (Organisation des Nations Unies)** : Rapports sur les crises humanitaires
- **OMS (Organisation Mondiale de la Santé)** : Données sur les pandémies et épidémies
- **UNICEF** : Rapports sur les crises affectant les enfants
- **Croix-Rouge / Croissant-Rouge** : Rapports sur les catastrophes

## Notes importantes

- Toutes les données sont basées sur des sources officielles et vérifiables
- Les coordonnées géographiques (latitude/longitude) proviennent des rapports officiels ou des bases de données géographiques
- Les populations touchées sont des estimations basées sur les rapports officiels des organisations internationales
- Les dates correspondent aux dates réelles documentées des événements
- Les crises marquées "en_cours=True" sont des crises actives en 2024-2025 selon les dernières données disponibles
